 Construction Production Management System.

A web-based system developed to manage and track daily production activities in a construction environment. This system replaces manual data handling with an automated digital solution.


🚀 Features
- 📊 Daily production data tracking  
- ⏱️ Automatic calculation of working hours  
- 📈 Production summary generation  
- 🧾 Data management and reporting  
- 🎨 User-friendly interface  


🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  

🎯 Purpose
This project was developed to improve efficiency in construction site data management by replacing manual processes with an automated system.
